Step 1 — Paste a YouTube link
Heads‑up: This UI doesn’t download anything by itself. You must connect it to your own backend (e.g., a small service that wraps yt‑dlp) and only download content when you have rights/permission.
Developer notes · API contract (connect your backend)

Point these endpoints to your self‑hosted service (Node, Python, etc.) that uses yt-dlp (or similar) under the hood. 4K often ships as video‑only streams; your backend should mux with audio (ffmpeg) when needed.

  • GET /api/formats?url=<VIDEO_URL>&maxQuality=2160 → returns JSON with a formats array.
  • GET /api/download?id=<formatId>&url=<VIDEO_URL> → returns a file stream (set Content-Disposition).
